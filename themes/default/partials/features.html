Qué puedes hacer con Zum

  • Crear sitios rápidos usando solo HTML, CSS y JavaScript
  • Cargar colecciones desde archivos JSON y renderizarlas dinámicamente
  • Definir plantillas en `js/config.js` y reutilizar partials
  • Construir y cambiar themes en `themes/<name>/` con `theme.css` y `partials/`
